# 回声之境 | Echoesphere
在未来，人们与虚拟世界的交互方式将不仅停留在传统的键盘鼠标输入，而会拥有更多的交互类型。本项目将探索一种由人工智能驱动的，富有趣味性的新型游戏交互方式，结合实体的按钮装置和玩家情绪，在游戏中借助做出相应的图形图像变化，并以此作为核心的游戏交互手段来设计游戏关卡。

## 项目说明
这是我本科的毕业设计。目前该项目尚在文献搜集和开题准备阶段，待开题后我将公开项目路线图以及 PDF 格式的概念设计稿。

此仓库是 Unity 工程。  
如想查看项目内容，请首先满足以下条件：
1. 安装了 Unity 6000.2.12f1 版本
2. 启用了 git LFS 功能

此游戏未来展示时预计会采用屏幕+实体装置交互的方式，预计通过 TCP Socket 与树莓派实时通信。见子项目：[echoesphere-communication](https://github.com/AMagicPear/echoesphere-communication)。
